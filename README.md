## 🧠 О проекте

**Gjallarbru** — это приложение, предназначенное для удобного чтения и сравнения оригинальных текстов (например, древнескандинавских поэм) с переводами.

Проект решает ключевую проблему:  
👉 разные переводы не совпадают по структуре  

Поэтому:
- тексты синхронизированы вручную
- структура строго выровнена по индексам
- UI учитывает различия между поэзией и прозой

---

## ✨ Основные возможности

### 📖 Чтение текстов
- оригинал (Old Norse)
- перевод на русский
- перевод на английский

### 📚 Типы отображения
- **Поэзия** → 2 колонки (оригинал / перевод)
- **Проза** → вертикальный формат (оригинал сверху, перевод снизу)

### 🌍 Интернационализация
- интерфейс: RU / EN
- тексты не зависят от i18n (контент отдельно)

### 🔍 Поиск
- fuzzy поиск через Fuse.js
- поиск по названиям и текстам

### 🎨 Темизация
- dark / light режим
- сохраняется у пользователя

### 📱 UX
- адаптивный интерфейс
- infinite scroll
- быстрые переходы
- минимальные перерендеры
